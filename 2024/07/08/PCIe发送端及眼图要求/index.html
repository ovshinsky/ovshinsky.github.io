发送端 （32.0 GT/S）模块要求

符号 参数描述 参数值 单位 备注
UI (Tx) 单位区间 31.246875~31.253125 ps 不包括 SSC 变化
BW_(TX-PKG-PLL1) 与 PKGTX-PLL1 对应的发送（Tx）锁相环（PLL）带宽 0.5 ~ 1.8 MHz 二阶锁相环（PLL）系统中的抖动传输界定函数
BW_(TX-PLL1) 与 BWTX-PKG-PLL1 相关的发送锁相环（PLL）的峰值特性 0 ~ 2 MHz 二阶锁相环（PLL）系统中的抖动传输界定函数
V_(TX-DIFF-PP) 全摆幅（full swing）操作条件下，差分发送（Tx）电压摆幅的峰-峰值（peak-peak） 800 ~1300 mVPP 在符合性测试负载条件下测量。定义为 2× | VTXD+−VTXD |
V_(TX-DIFF-PP-LOW) 低摆幅（low swing）操作条件下，差分发送（Tx）电压摆幅的峰-峰值（peak-peak） 400 ~1300 mVPP 在符合性测试负载条件下测量。定义为 2× | VTXD+−VTXD |
V_(TX-EIEOS-RS) 在减小摆幅信号传输期间，EIEOS 过程中的最小电压摆幅 > 232 mVPP “EIEOS” 指的是一种信号序列，用于表示在通信系统中从空闲状态到数据传输状态的过渡。
ps21_(TX-ROOT-DEVICE) 包含根端口时的ps21损耗 < 8.5 dB
ps21_(TX-NON-ROOT-DEVICE) 不包含根端口的所有设备的ps21损耗 < 3.7 dB
ILfit_(TX-ROOT-DEVICE) 在奈奎斯特频率处的拟合插入损耗 < 9.0 dB
ILfit_(TX-NON-ROOT-DEVICE) 在奈奎斯特频率处的拟合插入损耗 < 4.0 dB
V_(TX-BOOST-FS) 全摆幅时的最大名义发送器增益比率 > 8 dB 超过8.0 dB的名义增益被限制，以确保满足PS21TX
V_(TX-BOOST-RS) 减小摆幅时的最大名义发送器增益比率 > 2.5 dB
EQ_(TX-COEFF-RES) 发送端系数分辨率 1/63 ~ 1/24 -
T_(TX-UTJ) 发送端（Tx）不相关总抖动 < 6.25 psPP at 10-12
T_(TX-UTJ-SRIS) 在使用 SSC 测试其时钟模式时的发送端（Tx）不相关总抖动 < 7.15 psPP at 10-12
T_(TX-UDJDD) 非嵌入式参考时钟情况下的发送端（Tx）不相关抖动（Dj） < 3.125 psPP at 10-12
T_(TX-UPW-TJ) 总的不相关脉宽抖动 < 6.25 psPP at 10-12
T_(TX-UPW-DJDD) DjDD [确定性抖动] 不相关脉宽抖动 < 2.5 Ps PP
T_(TX-RJ) 发送的信号在时间上的随机变化 0.23 ~ 0.45 Ps RMS 这只是一个信息性参数。随机抖动（RJ）可能的范围从零到允许的最大 TTX-UDJDD
L_(TX-SKEW) 在多通道通信接口或系统中，不同通道输出之间的时序或相位对齐差异 1.25 ns 在单个发送器模块或单元内部，任意两个通道输出之间可能出现的时序或相位失调
RL_(TX-DIFF) 发送器封装加芯片差分回波损耗 见后
RL_(TX-CM) 发送器封装和芯片中测量的共模回波损耗 见后

通道眼图（32.0 GT/S）容忍度要求（Eye mask values)：

符号 参数描述 参数值 单位 备注
V_(RX-CH-EH-32G) 眼高 > 15 mVPP 误码率（BER）为10的负12次方时的眼高
V_(RX-CH-EW-32G) 零交叉点处的眼宽度 < 0.3 UI 误码率（BER）为10的负12次方时的眼宽
T_(RX-SAMPLE-OFFSET-32G) 从UI中心向左的最大采样位置偏移量 0.3 UI
T_(RX-SAMPLE-GRANULARITY-32G) 采样位置偏移的粒度 0.05 UI 可以改变或配置采样位置偏移的最小增量
V_(RX-DFE-D1-32G) DFE d1系数的范围 Abs(D1)/D0 [光标幅度] <= 0.8
V_(RX-DFE-D2-32G) DFE d2系数的范围 ±20 mV
V_(RX-DFE-D3-32G) DFE d3系数的范围 ±20 mV
