总体描述

ISG - D5680H是一个高性能8通道的53Gbaud/s差分输入/差分输出的MZ驱动芯片，可用于800Gbps的应用。驱动器可以支持的输入信号最高为600mVpp。在线性模式下，器件的最高输出电压摆幅为3.5Vpp，具有较低的总谐波失真（THD，Total Harmonic Distortion）。
ISG-D5680被设计为与调制器芯片进行DC耦合。芯片的尺寸为4.07×1.57mm。通道间隔为375μm。芯片提供了SPI的控制方式用于设置驱动增益和偏置电压。同时可以监测驱动芯片和调制器之间的直流共模电压，用于精确设置光子芯片调制器的偏置点。
芯片能够控制硅光芯片的加热器，并且能够运行两个无关的CW激光器，形成自动功率控制（APC）回路。
ISG-D5680H支持以下两种架构：

  • ISG-D5680H 倒装版本
  • ISG-D5680HW 引线键合封装版本

特点

  • 支持速率最高可达53Gbaud/s
  • 差分输入，差分输出
  • 射频输入：AC耦合；射频输出：DC耦合
  • 特征带宽为42GHz
  • 低功耗：360mW/channel @ 3Vpp（与Modulaor进行DC耦合，包括调制器负载）
  • 线性输出最高可达3.5Vpp
  • 通道间隔为375μm
  • 芯片尺寸为4.07mm×1.57mm
  • TX LOS dectect：发射链路信号丢失情况检测
  • 可调TX LOS阈值
  • 支持自动噪声抑制的工作模式
  • 调制器共模电压监测
  • SPI数字接口
  • 硅光芯片的热调控制
  • CW激光器的控制
  • MPD电流监测
  • 可调低频增益：12~19dB

高速低驱动电压硅基光子晶体慢光调制器

发表于 2024-09-09 更新于 2024-09-18
本文字数： 8k 阅读时长 ≈ 7 分钟

参考文献：高速低驱动电压硅基光子晶体慢光调制器 2024.08.

补充材料：高速低驱动电压硅基光子晶体慢光调制器-补充材料 2024.08.

摘要

硅光调制器在光学收发器中具有高密度集成和低制造成本的优势，但它们还需要提供高速和低功耗，以满足未来数据中心和高性能计算的需求。本文展示了一种显著改进的64 Gbps硅马赫–曾德尔调制器，该调制器结合了光子晶体慢光移相器。通过采用分布式电极并优化其阻抗，同时实现了电光相位匹配和电阻抗匹配，使驱动电压降低至0.87 V，这与鳍型场效应晶体管兼容，并且无需额外的电放大器。该调制器的比特能量低至59 fJ/bit，与微环调制器相当，同时由于其6 nm的宽工作谱，不需要像微环调制器那样进行温度控制。这些结果表明，该调制器在解决下一代数据基础设施中的功耗问题方面具有潜力。

背景介绍

人工智能和物联网的快速发展大大增加了对数据中心和高性能计算机的需求[1,2]。光互连被用于减少电气布线中的能量损失[3,4]。由于硅（Si）光子技术与成熟的电子工艺兼容，Si光子技术已成为光互连的一种强大的光子集成平台[5]。光学收发器中的硅光调制器具有高速、小尺寸和低制造成本等优点[6–14]。为了应对未来的节能问题，硅调制器必须同时实现高速和低功耗[15,16]。微环调制器（MRMs）在共振波长下可以实现小尺寸和低至几十fJ/bit的比特能量[17–20]。然而，外部温度控制为了抑制共振漂移，会增加总功耗[12–14]。相比之下，马赫–曾德尔调制器（MZMs）在宽波长范围内工作，因此无需温度控制，但其长移相器需要数pJ/bit的比特能量[21–26]。为了提高能效，已经研究了异质集成材料，如III-V族半导体[27,28]、聚合物[29,30]、石墨烯[31,32]和铌酸锂[33–36]。然而，这种集成需要复杂的制造工艺，与标准的互补金属氧化物半导体（CMOS）技术不兼容。

一个重要的研究问题是，硅马赫–曾德尔调制器（Si MZMs）能否通过结构创新来降低比特能量，而不涉及工艺变更。光子晶体波导（PCWs）表现出低群速度的慢光效应，这增强了任何光学效应[37]。慢光使得Vπ L与群折射率ng成比例减小，并降低了移相器的长度和电容Cpn[38]。然而，当将慢光应用于高速MZMs时，即使对于较短的移相器，也必须解决电光相位匹配问题，因为慢光和射频（RF）信号的群速度差异远大于通常的MZMs。为此，蛇形线RF电极被插入到PCW慢光MZM中的行波电极中，作为电延迟线，已证明其操作速度可达64 Gbps[39,40]。然而，驱动电压Vpp仍然高达3.5–5.2 V，这无法通过鳍型场效应晶体管（FinFET）提供，因而需要额外的电放大器。

与此相关的是，比特能量的减少是另一个关键问题。在先前的研究中，为评估移相器的调制能力，比特能量是通过在p-n结中积累的电荷能量CpnVp²/2来估算的，并且部分能量被结电阻Rpn消耗。然而，在行波MZM中的实际功耗主要由终端电阻RT决定。当Rpn可以忽略不计时，这一点很容易理解，因为p-n结中不会发生功耗。如果Rpn不可忽略，结果几乎相同；尽管一些射频（RF）功率在结处损失，剩余的功率最终还是在RT中被消耗。根据实际功耗Vp²/4RT，先前慢光MZMs的比特能量在25、32和64 Gbps的比特率下分别为200、625和2,400 fJ/bit【39,40】。在64 Gbps下的高比特能量是由于高驱动电压和20Ω的低RT，后者被用来改善高频响应。** 通过增加RT可以降低比特能量，但为了实现阻抗匹配，电极的特征阻抗Z0也必须增加。如果寄生电阻较小，Z0可近似为√(L/Cpn)。 为了增加电感分量L，螺旋电感的分布式电极被引入到一种肋型MZM中【41】。 然而，肋型移相器的Cpn非常大，因此需要大的电感器来获得高Z0，这导致了大的占用面积和RF损耗，将带宽限制在20 GHz以下。 **另一方面，将这一概念应用于慢光MZM，缩短移相器长度导致的小Cpn的和合理长度的电极电感可以实现高Z0，尽管在之前的MZM中还没有考虑到这一点。

在本研究中，我们设计了慢光马赫–曾德尔调制器（MZM）中分布式电极的阻抗，使电光相位匹配和阻抗匹配同时得以实现。本文详细介绍了其设计、制造过程以及在64 Gbps下的演示。最终，我们得出结论，该器件在平衡高速、低电压、低比特能量、小占用面积以及无需温度控制的操作方面具有很大的潜力，是一个很有前途的候选方案。

图1(f)和图1(g)显示了为了实现Z0 = 50Ω所需的电极总长度和电光带宽f3 dB的计算结果。随着群折射率ng的增加，用于实现高阻抗的电极变得更短，从而减少射频（RF）损耗。因此，慢光效应与分布式电极的结合可以在保持宽带宽的同时实现高阻抗，从而降低比特能量。在足够高的ng下，f 3dB接近移相器的固有RC限制。然而，光子晶体波导（PCW）的工作谱也会变窄。为了在足够的电光带宽和足够的工作谱之间实现最佳平衡，我们选择了ng = 20。

分布式电极的另一个优势是它们在慢光和射频（RF）信号之间的相位匹配，这一点已在先前的蛇形线电极中得到验证[39]。在本研究中的分布式电极中，较大的N使得相位匹配更加严格，并且提供了更高的sinc滚降频率。每个单元电路的相位移相由θU(ω) = βRFlU给出。图2(a)和图2(b)展示了标准电极和分布式电极的电光转换增益和群延迟的模拟频率响应。这些模拟使用了Keysight的高级设计系统（ADS）电路模拟器和内部光子设备模型进行。模型中包括电光相位不匹配、RC时间常数和RF传播损耗等作为带宽限制因素。这些模拟结果与测量结果至少达到40 GHz一致[42]。类似于蛇形线电极，分布式电极成功地抑制了sinc滚降，导致宽带响应，f3dB = 39 GHz。模拟得到的f3dB略低于图1(g)中的计算值，这主要是由于配线中的串联寄生电阻引起的额外RF损耗，但对于无需数字信号处理（DSP）的64-Gbaud信号传输而言，已经足够。该调制器主要的带宽限制因素是p-n结的RC时间常数，频率响应表现出可以视为一阶滞后元件的缓慢滚降。由于滚降缓慢，可以通过应用简单的线性均衡器进一步提高带宽。例如，仅以1.3 dB的直流抑制，f3dB可以达到50 GHz，并且能够传输100-Gbaud信号。这一特性突显了所提议的MZM在未来光互连中的潜在应用。此外，由于严格的阻抗匹配，群延迟波动在40 GHz以内保持在±1 ps范围内。平坦的群延迟响应减少了抖动并改善了眼图开口。分布式电极的一个重要特性是阻抗和相位延迟完全独立。电极设计参数（ZH和lm）可以独特地确定，以同时满足阻抗匹配和电光相位匹配要求。更多详细的分析和设计指南见补充材料1的第2节。

这些所提议MZM的结构优势可以应用于其他材料平台，只要它们的移相器得到缩短且结电容得到减少，例如通过慢光效应。为了获得高阻抗所需的电极电感也相应减少，从而实现实际的小尺寸和低RF损耗。例如，最近有报告了带有慢光增强的薄膜铌酸锂（TFLN）MZM[43]。TFLN MZM需要更小的电极间隙，以获得更小的Vπ，同时还需要阻抗匹配、相位匹配和低RF损耗[44]。将本研究中采用的电极工程应用于TFLN慢光MZM，将有助于克服这些权衡。

结果和讨论

器件加工

图3(a)展示了在兼容CMOS的300毫米硅绝缘体（SOI）工艺中制造的原型设备。其占地面积仅为0.5 mm × 0.6 mm = 0.3 mm²，包括电极。通过电磁场分析，将共面波导的导线宽度和间隙设置为8 μm和6 μm，以实现ZH = 55Ω。为了抑制高频损耗，硅虚拟层被阻塞在共面波导正下方（见补充材料1）。图3(b)和图3©显示了PCW移相器的放大视图和示意图。晶格常数设置为a = 404 nm，孔直径为d = 230 nm，以实现C波段光子带隙。p型和n型掺杂浓度分别设置为NA = 9.5 × 10¹⁷ cm⁻³和ND = 1.0 × 10¹⁸ cm⁻³，以在光学损耗和调制效率之间取得平衡。电极接触底部高度掺杂，以实现优良的欧姆接触。图3(d)展示了设备的测量光学传输谱。通过将测量谱归一化到参考硅波导的谱，去除了测量设置中的固有损耗。总插入损耗约为10 dB，包括多模干涉耦合器的0.5 dB × 2、硅波导与PCW之间的过渡0.4 dB × 2、弯曲0.03 dB × 6，以及p-n掺杂PCW移相器的约8 dB。由于工艺变化，板厚度略微减小，这略微提高了光学损耗并使光谱窗口向短波长方向移动。然而，6 nm的光谱窗口足够宽，可以消除温度补偿。通过采用之前展示的晶格偏移PCWs，可以覆盖整个C波段[45]。未来，预计通过优化掺杂轮廓[46]和组件结构[47]，以及研究新材料[43]，可以降低损耗。该原型设备在下述调制实验中使用的波长下显示ng ≈ 30，这比目标ng = 20略高，原因是板厚度的变化（见补充材料1的第5节）。这种变化导致电光相位不匹配，将带宽缩小到f3dB ≈ 31 GHz，但仍保持了64-Gbaud调制能力。

阻抗

分布式电极的阻抗通过图4(a)中的设置进行了评估。使用了Keysight PNA-X N5247B矢量网络分析仪，通过在探针尖端的线反射反射匹配方法进行了校准。测量得到的S参数经过去嵌入处理，以去除焊盘电容，确保与模拟结果的一致性。设备通过bias-T施加了直流偏置电压VDC = −2 V。图4(b)展示了测量和模拟的反射系数。测量结果与模拟结果大致一致。模拟与测量之间的偏差源于电磁场分析中的模型误差。所有频率下的反射系数均小于−18 dB，达到了针对50Ω参考阻抗的令人满意的宽带匹配。图4©展示了输入阻抗的实部和虚部。在测量频率范围内（高达60 GHz），电阻分量和电抗分量分别接近50 Ω和0 Ω。这种频率无关的高阻抗降低了功耗，同时保持了优良的信号质量并降低了比特能量。

背靠背的信号传输

原型设备的传输性能进行了实验评估。使用了连续波可调激光器作为光源，并将马赫–曾德尔干涉仪的偏置调整到正交点（quad. bias）或从正交点下调5 dB（deep bias）。脉冲模式发生器（PPG）和多路复用器生成了50 Gbps或64 Gbps的非归零（NRZ）信号。设备由宽带放大器推拉驱动，驱动电压分别为Vpp = 0.87 V、1.6 V和2.2 V。前两个电压与12 nm FinFET CMOS工艺的核心和IO电压兼容[48]。使用采样示波器监测接收信号，并通过离线数字信号处理（DSP）中的五段线性均衡器进行均衡。实验细节见附录A。

图5(a)展示了50-Gbaud无DSP眼图和预期的比特错误率（BER）。在2.2 V驱动电压下，正交点（quad. bias）和深偏置（deep bias）的动态消光比（ERs）分别为2.0 dB和4.5 dB。正交点和深偏置条件下的调制损耗分别为2.1 dB和6.3 dB。观察结果显示，50-Gbps无DSP的清晰眼图在Vpp < 1 V时都能实现。RF信号功耗在Vpp = 0.87 V、1.6 V和2.2 V时分别为3.8 mW、13 mW和24 mW。尽管功耗低，但预计的BER在所有驱动电压下都达到了前向纠错（FEC）阈值。高阻抗电极成功地降低了驱动电流，比特能量分别达到76 fJ/bit、256 fJ/bit和484 fJ/bit。这些结果远低于标准硅MZM的几皮焦耳（pJ/bit）。图5(b)展示了64-Gbaud无DSP眼图和估计的BER。在64 Gbaud下，眼图略有退化，但接收信号仍然满足Vpp > 1.5 V的软决策（SD）FEC阈值。图5 ©和图5(d)展示了应用五段线性均衡器的50和64-Gbaud眼图。由于模拟预期的慢滚降响应，简单的线性均衡器显著提高了信号质量。该设备在Vpp = 0.87 V下有效传输了64-Gbps信号，展示了优良的FinFET兼容性，比特能量为59 fJ/bit。

表1总结了最近在CMOS兼容硅调制器方面的进展。为了估算我们设备的调制效率Vπ L，我们在高驱动电压Vpp = 3.17 V下进行了2.4-Gbps调制实验，以最小化滚降和噪声。设备由PPG直接驱动，并在正交点进行偏置。我们测得ER = 4.19 dB，Vπ L估算为每个移相器0.32 V·cm，或推拉驱动情况下为0.16 V·cm。考虑到由于高频响应导致的滚降，DC Vπ L可能会小于此值。在我们之前的实验[40]中，ng = 20时测得DC Vπ L为0.44 V·cm。因此，鉴于原型设备的ng = 30，上述值是合理的。传统上，超过50 Gbps的MZM需要更小的RT[25,50,51]。使用50Ω终端的先前研究面临比特率的显著限制[41,54,55,57]。单驱动结构导致驱动电压达到或超过5 V，比特能量超过1.5 pJ/bit[21,26,54]。本研究展示了在64 Gbps下比特能量低于100 fJ/bit，克服了效率和比特率之间的权衡，如图6所示。比特能量接近MRM的水平，同时具有6 nm的工作光谱，这远宽于MRM的典型亚1-nm范围[18,19]。

结论

在这篇论文中，展示了使用硅（Si）光子晶体波导（PCW）慢光马赫–曾德尔调制器（MZM）在低于1 V的驱动电压下进行64-Gbps的背对背信号传输。分布式电极同时实现了50Ω阻抗和电光相位匹配，适度地抑制了RF损耗。这种方法突显了硅慢光MZM在功耗方面的潜在优势，实现了59 fJ/bit的比特能量。宽达6 nm的工作带宽消除了对温度控制的需求，从而提升了整体系统的功率性能。此外，设备的占地面积减少至0.3 mm²，同时保持了CMOS工艺兼容性。所提出的MZM的结构优势可以应用于其他材料平台，并预计会有进一步的性能提升。这些结果表明，硅慢光MZM有助于解决未来数据中心和高性能计算中的功率问题。
